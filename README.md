# Python Fundamentals

A structured collection of **Python fundamentals, exercises, and learning notes** designed to build a strong programming foundation.

This repository documents my journey of learning Python by implementing core programming concepts, experimenting with code, and organizing knowledge through **Jupyter notebooks and Python scripts**.

The goal is to create a **well-structured reference for Python basics** that can also help beginners understand how fundamental programming concepts work in practice.

---

## Repository Structure

| Folder | What it Covers |
| --- | --- |
| [`loops/`](./loops) | `for` loops, `while` loops, iteration patterns, and step-by-step examples |
| [`Functions/`](./Functions) | Defining functions, parameters, return values, and function-based exercises |
| [`Lambda_functions/`](./Lambda_functions) | Lambda expressions, anonymous functions, and use with `map`, `filter`, `sorted` |
| [`list/`](./list) | List creation, indexing, slicing, and common list operations |
| [`Tuples/`](./Tuples) | Tuple basics, immutability, and practical use cases |
| [`Dictionary/`](./Dictionary) | Dictionary creation, access, iteration, and manipulation |
| [`OOP/`](./OOP) | Classes, objects, inheritance, encapsulation, and OOP design patterns |
| [`Advance Python/Decorators/`](./Advance%20Python/Decorators) | Function decorators, `@wraps`, and real-world decorator patterns |
| [`string_fundamentals/`](./string_fundamentals) | String methods, formatting, slicing, and common operations |
| [`Notebooks/`](./Notebooks) | Jupyter notebooks with interactive examples and concept explanations |
| [`Practical_application/`](./Practical_application) | Small projects applying core Python concepts end-to-end |
| [`go-python-integration/`](./go-python-integration) | Calling Python from Go — cross-language integration examples |

### Root-level scripts

## What This Repository Focuses On

This project focuses on building **strong programming fundamentals**, including:

- Python syntax and structure
- Writing clean and readable functions
- Understanding loops and control flow
- Working with common data structures (lists, tuples, dictionaries)
- Object-oriented programming principles
- Logical problem solving and practical applications

The repository is structured to move **from basic concepts toward practical coding exercises and real-world patterns**.

## Technologies Used

- **Python 3** — core language

- **Go** — used in the cross-language integration examples

## Why This Repository Exists

Many beginners jump directly into advanced topics like **machine learning or frameworks** without building strong fundamentals.

This repository focuses on:

- Strengthening core programming skills
- Practicing logical thinking
- Creating a structured learning path for Python

A strong foundation makes it easier to learn **data science, machine learning, and systems programming later**.

---

## How to Use This Repository

1. Start with [`loops/`](./loops), [`Functions/`](./Functions), and [`list/`](./list) if you're new to Python
2. Move into [`Dictionary/`](./Dictionary), [`Tuples/`](./Tuples), and [`OOP/`](./OOP) once you're comfortable with the basics
3. Explore [`Advance Python/Decorators/`](./Advance%20Python/Decorators) for more advanced patterns
4. Check the [`Notebooks/`](./Notebooks) folder for interactive walkthroughs
5. Look at [`Practical_application/`](./Practical_application) and the root scripts to see concepts applied in real code
